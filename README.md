# hardware-rental-17

This repository consists of multiple parts. The contents of the hardwarerental folder are the Hardware Rental Moodle Plugin. The contents of the hardwarerental-bpx folder
include all of the files that are related to the Camunda BPX engine, including Java files and BPX files and the deployable war file.
The contents of the digitalsignature folder are the Digital Signature Moodle plugin developed by the previous SWIM team, modified by us.
The contents of the digitalsignature_web folder are the Digital Signature webservice, developed by the previous SWIM team.
The repository also includes the zip file which is needed for the DS webservice. It can be found online, but it is included to make this repository complete and usable on its own.
